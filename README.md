# Sistem Booking Laboratorium

Sistem manajemen dan pemesanan laboratorium menggunakan React, TypeScript, Tailwind CSS, dan DaisyUI.

## Teknologi yang Digunakan

- React 18 + Vite
- TypeScript
- Tailwind CSS
- DaisyUI
- ESLint untuk kode yang konsisten
